# SLAs de TI - Notebooks

Este documento define os tempos de SLA para problemas relacionados a notebooks no setor de TI.

## Mapeamento de Prioridades

### Incidentes (Problemas Técnicos)

**P1 - 1 hora (60 minutos)**
Problemas críticos que impedem o trabalho:
- Notebook travando
- Notebook não liga

### Requisições (Solicitações de Serviço)

**P2 - 4 horas (240 minutos)**
Solicitações urgentes:
- Devolver notebook
- Configurar notebook

**P3 - 24 horas (1440 minutos / 1 dia)**
Solicitações de médio prazo:
- Solicitar troca de notebook

**P4 - 48 horas (2880 minutos / 2 dias)**
Solicitações de longo prazo:
- Solicitar novo notebook

## Configuração no Sistema

Os itens foram cadastrados na subcategoria "Notebooks" (ID: 67) com os tempos de SLA documentados.

Para aplicar estes SLAs:
1. Problemas técnicos (travando, não liga): usar tipo "Incidente" com prioridade P1
2. Solicitações (novo, troca, devolução, configuração): usar tipo "Requisição" com prioridade P2, P3 ou P4
3. O sistema calculará automaticamente o prazo baseado no tipo e prioridade selecionados
