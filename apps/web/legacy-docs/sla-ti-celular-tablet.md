# SLAs de TI - Celular/Tablet

Este documento define os tempos de SLA para problemas relacionados a celulares e tablets corporativos no setor de TI.

## Mapeamento de Prioridades

### Incidentes (Problemas Técnicos)

**P1 - 1 hora (60 minutos)**
Problemas críticos e urgentes:
- Celular/tablet travando
- Celular/tablet não funciona
- Devolver celular/tablet

**P2 - 2 horas (120 minutos)**
Configurações e formatação:
- Configurar celular/tablet
- Formatar celular/tablet

### Requisições (Solicitações de Serviço)

**P4 - 48 horas (2880 minutos / 2 dias)**
Solicitações de novos equipamentos:
- Solicitar novo celular/tablet
- Solicitar troca de celular/tablet

## Configuração no Sistema

Os itens foram cadastrados na subcategoria "Celular/Tablet" (ID: 86) com os tempos de SLA documentados.

Para aplicar estes SLAs:
1. Problemas urgentes (travando, não funciona, devolver): usar tipo "Incidente" com prioridade P1
2. Configurações (configurar, formatar): usar tipo "Incidente" com prioridade P2
3. Solicitações (novo, troca): usar tipo "Requisição" com prioridade P4
4. O sistema calculará automaticamente o prazo baseado no tipo e prioridade selecionados
