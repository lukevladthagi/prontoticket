# SLAs de TI - Impressoras

Este documento define os tempos de SLA para problemas relacionados a impressoras no setor de TI.

## Mapeamento de Prioridades

### P1 - 1 hora (60 minutos)
Problemas críticos que impedem completamente o trabalho:
- Troca de toner
- Troca impressora de etiquetas
- Acesso à impressora
- Troca ribbon
- Papel atolado
- Impressão fora do padrão
- Não imprime
- Configurar scanner
- Scanner não funciona
- Scanner fora do padrão

### P2 - 2 horas (120 minutos)
Problemas importantes mas com workarounds:
- Realocar uma impressora
- Impressora offline
- Não imprime na impressora correta

### P3 - 48 horas (2880 minutos / 2 dias)
Requisições de recursos e atividades planejadas:
- Solicitar recurso (toner)
- Receber recurso (toner)

## Configuração no Sistema

Os itens foram cadastrados na categoria "Impressora" (ID: 14) com os tempos de SLA documentados.

Para aplicar estes SLAs:
1. Ao criar um chamado relacionado a impressoras, selecione a prioridade adequada
2. O sistema calculará automaticamente o prazo baseado nos SLAs configurados
3. Itens P1: 1 hora de resolução
4. Itens P2: 2 horas de resolução  
5. Itens P3: 48 horas de resolução
