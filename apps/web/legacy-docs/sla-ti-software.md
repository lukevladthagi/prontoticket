# SLAs de TI - Serviços de Software

Este documento define os tempos de SLA para serviços de software no setor de TI.

## Mapeamento de Prioridades

### Incidentes (Problemas Técnicos)

**P2 - 2 horas (120 minutos)**
Todos os serviços de software:
- Instalação de novo software
- Backup
- Pacote Office
- Formatação
- Ajuste de software
- Configuração de software
- Instalar antivírus
- Configurar antivírus

## Configuração no Sistema

Os itens foram cadastrados na subcategoria "Serviços de Software" (ID: 135) com os tempos de SLA documentados.

Para aplicar estes SLAs:
1. Todos os serviços de software (2h): usar tipo "Incidente" com prioridade P2
2. O sistema calculará automaticamente o prazo baseado no tipo e prioridade selecionados
