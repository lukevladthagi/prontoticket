# SLAs de TI - E-mail

Este documento define os tempos de SLA para serviços de correio eletrônico no setor de TI.

## Mapeamento de Prioridades

### Incidentes (Problemas Técnicos)

**P1 - 1 hora (60 minutos)**
- Criar e-mail
- Alterar e-mail
- Redefinir senha e-mail

**P2 - 2 horas (120 minutos)**
- Configurar e-mail

**P2 - 4 horas (240 minutos)**
- Caixa de e-mail cheia
- E-mail sem acesso

## Configuração no Sistema

Os itens foram cadastrados na subcategoria "E-mail" (ID: 11) com os tempos de SLA documentados.

Para aplicar estes SLAs:
1. Gerenciamento de contas (1h): usar tipo "Incidente" com prioridade P1
2. Configuração (2h): usar tipo "Incidente" com prioridade P2
3. Problemas de acesso e quota (4h): usar tipo "Incidente" com prioridade P2
4. O sistema calculará automaticamente o prazo baseado no tipo e prioridade selecionados
