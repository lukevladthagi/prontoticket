# SLAs de TI - TV

Este documento define os tempos de SLA para problemas relacionados a televisores no setor de TI.

## Mapeamento de Prioridades

### Incidentes (Problemas Técnicos)

**P2 - 2 horas (120 minutos)**
Todos os problemas de TV:
- Ajuste de mídia TV
- TV sem conexão
- TV sem imagem
- TV não liga

## Configuração no Sistema

Os itens foram cadastrados na subcategoria "TV" (ID: 130) com os tempos de SLA documentados.

Para aplicar estes SLAs:
1. Todos os problemas de TV (2h): usar tipo "Incidente" com prioridade P2
2. O sistema calculará automaticamente o prazo baseado no tipo e prioridade selecionados
