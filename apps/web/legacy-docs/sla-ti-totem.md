# SLAs de TI - Totem

Este documento define os tempos de SLA para problemas relacionados a totens de atendimento no setor de TI.

## Mapeamento de Prioridades

### Incidentes (Problemas Técnicos)

**P1 - 1 hora (60 minutos)**
Problemas urgentes que impedem o funcionamento:
- Configurar totem
- Totem não funciona
- Totem não liga
- Totem sem internet
- Totem sem imprimir etiqueta

**P2 - 4 horas (240 minutos)**
Operações de remoção:
- Remover totem

### Requisições (Solicitações de Serviço)

**P3 - 48 horas (2880 minutos / 2 dias)**
Aquisição de novos equipamentos:
- Solicitar novo totem

## Configuração no Sistema

Os itens foram cadastrados na subcategoria "Totem" (ID: 112) com os tempos de SLA documentados.

Para aplicar estes SLAs:
1. Problemas urgentes (1h): usar tipo "Incidente" com prioridade P1
2. Remoção de totem (4h): usar tipo "Incidente" com prioridade P2
3. Novo totem (48h): usar tipo "Requisição" com prioridade P3
4. O sistema calculará automaticamente o prazo baseado no tipo e prioridade selecionados
